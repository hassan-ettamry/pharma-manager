# PharmaManager Frontend

Frontend application for pharmacy management system built with React and Vite.

---

## 🚀 Tech Stack

- React (Vite)
- Axios
- React Router

---

## ⚙️ Installation

cd pharma_frontend
npm install
npm run dev

## 🌐 Application URL

http://localhost:5173

## 🔗 Backend API

Make sure backend is running:

http://127.0.0.1:8000

Swagger documentation:

http://127.0.0.1:8000/api/schema/swagger-ui/

## 📁 Project Structure

src/
  api/         → API calls (Axios)
  hooks/       → business logic
  components/  → reusable UI components
  pages/       → application pages

## ✅ Features

- View medicaments
- Add medicament
- Delete medicament (soft delete)
- Create sales
- Cancel sales (restore stock)
- Dashboard with statistics

## ⚠️ Notes

- Ensure backend is running before starting frontend
- API base URL is configured in axiosConfig.js
